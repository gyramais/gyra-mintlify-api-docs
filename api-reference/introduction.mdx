# Conheça a GYRA+

A GYRA+ é a plataforma mais completa de análise de crédito automatizada do Brasil.  
Unificando dados públicos, informações de bureau, histórico judicial, relacionamentos e muito mais, nossa API permite que você integre inteligência avançada de crédito ao seu sistema com facilidade.

Ideal para fintechs, distribuidoras, bancos, factorings e qualquer empresa que queira **automatizar a análise de crédito com mais produtividade e menos risco**.

## Ambiente de Produção

Utilize a URL abaixo para acessar o ambiente de produção da GYRA+:

```url
https://gyra-core.gyramais.com.br/
```

## Tipos de Relatórios

Conheça os diferentes tipos de relatórios disponíveis e escolha o mais adequado para sua operação:

- **COMPLETO+**: relatório mais completo da GYRA+, com certidões públicas, processos desde 1995, protestos e tudo que é possível integrar.  
- **COMPLETO**: inclui dados profundos, mas sem certidões e com processos desde 2014.
- **ESSENCIAL**: visão enxuta sem certidões e protestos, mas com relacionamentos e dados principais.
- **SIMPLES**: apenas os dados cadastrais básicos e localização, ideal para qualificação rápida de leads ou limpeza de base.
