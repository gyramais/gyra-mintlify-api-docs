### Quando usar

Use este endpoint para gerar insight interpretativo sobre o relatório de crédito. Você pode usar quando quiser acelerar leitura analítica, produzir resumo executivo e apoiar decisão humana com explicações estruturadas.

### Autenticação

Bearer JWT no header `Authorization`.

### Corpo da requisição

```json
{
  "sectionId": "string"
}
```

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/insight' \
--header 'Authorization: Bearer {seu_token_jwt}' \
--header 'Content-Type: application/json' \
--data-raw '{
  "sectionId": "string"
}'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/insight", {
  method: "POST",
  headers: {
    "Authorization": "Bearer {seu_token_jwt}",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
  "sectionId": "string"
})
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/insight"
headers = {
  "Authorization": "Bearer {seu_token_jwt}"
}
payload = {
  "sectionId": "string"
}
response = requests.post(url, headers=headers, json=payload)
print(response.json())
```
