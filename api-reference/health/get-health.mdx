### Quando usar

Use este endpoint para checagem rápida de disponibilidade da API. Você pode usar em monitoramento básico, probes de infraestrutura e validação de conectividade antes de iniciar processamentos críticos.

### Autenticação

Sem autenticação obrigatória para esta operação.

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/health'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/health", {
  method: "GET",
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/health"
headers = {}
response = requests.get(url, headers=headers)
print(response.json())
```

</RequestExample>
