### Quando usar

Use este endpoint para listar as credenciais de integração já configuradas na organização. Você pode usar em telas de administração, auditoria operacional ou antes de iniciar alterações em integrações existentes.

### Autenticação

Bearer JWT no header `Authorization`.

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/integration/credential' \
--header 'Authorization: Bearer abc123'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/integration/credential", {
  method: "GET",
  headers: {
    "Authorization": "Bearer abc123",
  },
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/integration/credential"
headers = {
  "Authorization": "Bearer abc123"
}
response = requests.get(url, headers=headers)
print(response.json())
```

</RequestExample>

<ResponseExample>

```json 500 Internal Server Error
{
  "code": 500,
  "message": "Internal server error"
```

