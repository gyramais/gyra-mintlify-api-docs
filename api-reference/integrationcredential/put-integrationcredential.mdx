openapi: put /integration/credential/{id}
title: Atualizar Requisição
description: Atualiza dados deste recurso.
---

### Quando usar

Use este endpoint para atualizar credenciais já cadastradas. Você pode usar em rotação de chaves, troca de parâmetros de autenticação ou ajuste de configuração após mudança no provedor externo.

### Autenticação

Bearer JWT no header `Authorization`.

### Parâmetros

| Nome | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `id` | `path` | `string` | `sim` | Parâmetro da requisição. |

### Corpo da requisição

```json
{
  "id": "string"
}
```

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/integration/credential/{id}' \
--header 'Authorization: Bearer abc123' \
--header 'Content-Type: application/json' \
--data-raw '{
  "id": "string"
}'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/integration/credential/{id}", {
  method: "PUT",
  headers: {
    "Authorization": "Bearer abc123",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
  "id": "string"
})
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/integration/credential/{id}"
headers = {
  "Authorization": "Bearer abc123"
}
payload = {
  "id": "string"
}
response = requests.put(url, headers=headers, json=payload)
print(response.json())
```

</RequestExample>
