### Quando usar

Use este endpoint para cadastrar uma nova credencial de integração externa (bureau, provedor ou serviço parceiro). Você pode usar quando conectar a organização a um novo provedor de dados de crédito, configurando os dados necessários para consumo nas análises.

### Autenticação

Bearer JWT no header `Authorization`.

### Corpo da requisição

```json
{
  "policyId": "string",
  "document": "string",
  "type": "string"
}
```

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/integration/credential' \
--header 'Authorization: Bearer abc123' \
--header 'Content-Type: application/json' \
--data-raw '{
  "policyId": "string",
  "document": "string",
  "type": "string"
}'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/integration/credential", {
  method: "POST",
  headers: {
    "Authorization": "Bearer abc123",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
  "policyId": "string",
  "document": "string",
  "type": "string"
})
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/integration/credential"
headers = {
  "Authorization": "Bearer abc123"
}
payload = {
  "policyId": "string",
  "document": "string",
  "type": "string"
}
response = requests.post(url, headers=headers, json=payload)
print(response.json())
```
