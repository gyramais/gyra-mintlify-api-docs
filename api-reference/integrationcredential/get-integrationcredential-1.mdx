### Quando usar

Use este endpoint para consultar os detalhes de uma credencial específica. Você pode usar quando for editar, validar configuração ou diagnosticar falhas de integração de um provedor em particular.

### Autenticação

Bearer JWT no header `Authorization`.

### Parâmetros

| Nome | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `id` | `path` | `string` | `sim` | Parâmetro da requisição. |

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/integration/credential/{id}' \
--header 'Authorization: Bearer abc123'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/integration/credential/{id}", {
  method: "GET",
  headers: {
    "Authorization": "Bearer abc123",
  },
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/integration/credential/{id}"
headers = {
  "Authorization": "Bearer abc123"
}
response = requests.get(url, headers=headers)
print(response.json())
```

</RequestExample>
