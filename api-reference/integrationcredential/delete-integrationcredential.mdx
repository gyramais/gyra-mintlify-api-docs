openapi: delete /integration/credential/{id}
title: Remover Requisição
description: Remove um registro deste recurso.
---

### Quando usar

Use este endpoint para remover uma credencial de integração que não deve mais ser utilizada. Você pode usar quando descontinuar um provedor, revogar acesso comprometido ou limpar configurações obsoletas do ambiente.

### Autenticação

Bearer JWT no header `Authorization`.

### Parâmetros

| Nome | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `id` | `path` | `string` | `sim` | Parâmetro da requisição. |

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/integration/credential/{id}' \
--header 'Authorization: Bearer abc123'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/integration/credential/{id}", {
  method: "DELETE",
  headers: {
    "Authorization": "Bearer abc123",
  },
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/integration/credential/{id}"
headers = {
  "Authorization": "Bearer abc123"
}
response = requests.delete(url, headers=headers)
print(response.json())
```

</RequestExample>

<ResponseExample>

