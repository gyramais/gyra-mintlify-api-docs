### Quando usar

Use este endpoint em fluxos de exclusão de conta/dados (LGPD), antes da etapa definitiva de remoção. Você pode usar quando precisar validar a intenção do usuário e emitir um token temporário de confirmação para uma operação sensível de deleção.

### Autenticação

Bearer JWT no header `Authorization`.

### Corpo da requisição

```json
{
  "userIdToDelete": "string"
}
```

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/auth/user/create-deletion-token' \
--header 'Authorization: Bearer {seu_token_jwt}' \
--header 'Content-Type: application/json' \
--data-raw '{
  "userIdToDelete": "string"
}'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/auth/user/create-deletion-token", {
  method: "POST",
  headers: {
    "Authorization": "Bearer {seu_token_jwt}",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
  "userIdToDelete": "string"
})
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/auth/user/create-deletion-token"
headers = {
  "Authorization": "Bearer {seu_token_jwt}"
}
payload = {
  "userIdToDelete": "string"
}
response = requests.post(url, headers=headers, json=payload)
print(response.json())
```
