### Quando usar

Use este endpoint para acompanhar o ciclo de vida de um relatório específico e obter o resultado consolidado. Use após criar um relatório para fazer polling de status ou para buscar os dados finais antes de tomar decisão de crédito.

### Autenticação

Bearer JWT no header `Authorization`.

### Parâmetros

| Nome | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `id` | `path` | `string` | `sim` | ID do relatório retornado em `POST /v2/report`. |
| `tabType` | `query` | `string` | `não` | Tipo do relatório (filtra o conjunto de seções retornadas). |
| `details` | `query` | `boolean` | `não` | Define se os detalhes das seções devem ser retornados. Use `false` para resposta enxuta. |
