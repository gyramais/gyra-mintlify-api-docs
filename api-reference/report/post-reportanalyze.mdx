### Quando usar

Use este endpoint para disparar a etapa de análise de crédito com payload de processamento. Você pode usar em rotinas de processamento assistido, reprocessamento controlado ou fluxos em que a análise é acionada separadamente da criação inicial.

### Autenticação

Bearer JWT no header `Authorization`.

### Corpo da requisição

```json
{
  "id": "string",
  "approved": true,
  "commentary": "Aprovado após análise manual."
}
```

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/report/analyze' \
--header 'Authorization: Bearer abc123' \
--header 'Content-Type: application/json' \
--data-raw '{
  "id": "string",
  "approved": true,
  "commentary": "Aprovado após análise manual."
}'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/report/analyze", {
  method: "POST",
  headers: {
    "Authorization": "Bearer abc123",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
  "id": "string",
  "approved": true,
  "commentary": "Aprovado após análise manual."
})
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/report/analyze"
headers = {
  "Authorization": "Bearer abc123"
}
payload = {
  "id": "string",
  "approved": True,
  "commentary": "Aprovado após análise manual."
}
response = requests.post(url, headers=headers, json=payload)
print(response.json())
```
