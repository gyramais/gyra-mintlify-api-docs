```python Python
import requests

url = "https://gyra-core.gyramais.com.br/report"
headers = {
  "Authorization": "Bearer abc123",
  "Content-Type": "application/json"
}
payload = {
  "document": "12345678901",
  "type": "cpf",
  "policyId": "abc123"
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
```

### Quando usar

Use este endpoint para iniciar uma nova análise de crédito para CPF ou CNPJ. Você pode usar quando receber uma solicitação de avaliação de cliente e precisar gerar um relatório com base na política de decisão selecionada.

### Autenticação

Bearer JWT no header `Authorization`.

### Parâmetros

| Nome | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `front` | `query` | `string` | `sim` | Parâmetro da requisição. |

### Corpo da requisição

```json
{
  "policyId": "string",
  "document": "string",
  "type": "string"
}
```
