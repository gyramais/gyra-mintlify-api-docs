### Quando usar

Use este endpoint para recuperação técnica de relatórios com inconsistência de processamento. Você pode usar em cenários de correção operacional, retry controlado e estabilização de casos que ficaram incompletos por falha transitória.

### Autenticação

Bearer JWT no header `Authorization`.

### Corpo da requisição

```json
{
  "id": "string"
}
```

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/report/heal' \
--header 'Authorization: Bearer abc123' \
--header 'Content-Type: application/json' \
--data-raw '{
  "id": "string"
}'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/report/heal", {
  method: "POST",
  headers: {
    "Authorization": "Bearer abc123",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
  "id": "string"
})
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/report/heal"
headers = {
  "Authorization": "Bearer abc123"
}
payload = {
  "id": "string"
}
response = requests.post(url, headers=headers, json=payload)
print(response.json())
```
