### Quando usar

Use este endpoint para salvar dados de produto vinculados ao processo de decisão. Você pode usar quando a esteira exigir persistir configuração comercial/produto associada ao relatório para continuidade do fluxo de aprovação.

### Autenticação

Bearer JWT no header `Authorization`.

### Corpo da requisição

```json
{
  "id": "string"
}
```

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/report/save-product' \
--header 'Authorization: Bearer abc123' \
--header 'Content-Type: application/json' \
--data-raw '{
  "id": "string"
}'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/report/save-product", {
  method: "POST",
  headers: {
    "Authorization": "Bearer abc123",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
  "id": "string"
})
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/report/save-product"
headers = {
  "Authorization": "Bearer abc123"
}
payload = {
  "id": "string"
}
response = requests.post(url, headers=headers, json=payload)
print(response.json())
```
