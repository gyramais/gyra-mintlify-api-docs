### Quando usar

Use este endpoint para reanalisar um caso já processado anteriormente. Você pode usar quando houver mudança de política, atualização de dados do cliente ou necessidade de segunda opinião antes da decisão final.

### Autenticação

Bearer JWT no header `Authorization`.

### Corpo da requisição

```json
{
  "id": "string"
}
```

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/report/re-analyze' \
--header 'Authorization: Bearer abc123' \
--header 'Content-Type: application/json' \
--data-raw '{
  "id": "string"
}'
```

```javascript JavaScript
fetch("https://gyra-core.gyramais.com.br/report/re-analyze", {
  method: "POST",
  headers: {
    "Authorization": "Bearer abc123",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
  "id": "string"
})
})
  .then(res => res.json())
  .then(console.log)
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/report/re-analyze"
headers = {
  "Authorization": "Bearer abc123"
}
payload = {
  "id": "string"
}
response = requests.post(url, headers=headers, json=payload)
print(response.json())
```
