### Quando usar

Use este endpoint para acompanhar o ciclo de vida de um relatório específico e obter o resultado consolidado. Você pode usar após criar um relatório para fazer polling de status ou para buscar os dados finais antes de tomar decisão de crédito.

### Autenticação

Bearer JWT no header `Authorization`.

### Parâmetros

| Nome | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `id` | `path` | `string` | `sim` | Parâmetro da requisição. |
| `tabType` | `query` | `string` | `não` | Report type |
| `details` | `query` | `string` | `não` | Para remover os detalhes da seção, defina como `false` |
