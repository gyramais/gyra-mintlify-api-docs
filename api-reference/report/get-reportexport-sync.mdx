### Quando usar

Use este endpoint para gerar e receber o arquivo de export (PDF, XLS ou ambos) de um relatório **na mesma requisição**. A chamada bloqueia até o arquivo estar pronto e retorna direto a URL para download. Para volumes grandes ou quando preferir não bloquear, use `GET /report/export/{id}` com webhook `REPORT_EXPORTED`.

### Autenticação

Bearer JWT no header `Authorization`.

### Parâmetros

| Nome | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `id` | `path` | `string` | `sim` | ID do relatório a exportar. |
| `format` | `query` | `string` | `não` | Formato do arquivo: `pdf`, `excel` ou `both`. Padrão: `pdf`. |
| `type` | `query` | `string` | `não` | Tipo de export: `REPORT` (completo) ou `REPORT_MINIFIED` (resumido). Padrão: `REPORT`. |

```python Python
import requests

print(response.json())
```

</RequestExample>

<Tip>
O link retornado tem validade de 7 dias. Para geração em lote ou relatórios muito grandes, prefira a versão assíncrona (`GET /report/export/{id}`) para não segurar a conexão HTTP.
</Tip>
