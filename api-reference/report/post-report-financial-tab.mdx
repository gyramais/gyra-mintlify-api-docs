---
openapi: post /report/{id}/financial-tab
title: Enviar Balanço e DRE
description: Envia o arquivo de balanço e DRE para extração e análise em um relatório.
---

<Tip>
Enviar o arquivo de balanço e DRE via form_data no campo 'file'.
</Tip>

### Quando usar

Use este endpoint para processar ou atualizar a aba financeira vinculada a um relatório existente. Você pode usar quando precisar recalcular informações financeiras específicas após nova evidência, atualização cadastral ou ajuste de dados de entrada.

### Autenticação

Bearer JWT no header `Authorization`.

### Parâmetros

| Nome | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `id` | `path` | `string` | `sim` | Parâmetro da requisição. |

<RequestExample>

```bash cURL
curl --location 'https://gyra-core.gyramais.com.br/report/{id}/financial-tab' \
--header 'Authorization: Bearer abc123' \
--form 'file=@"/caminho/balanco-dre.pdf"'
```

```python Python
import requests

url = "https://gyra-core.gyramais.com.br/report/{id}/financial-tab"
headers = {
  "Authorization": "Bearer abc123"
}
files = {
  "file": open("/caminho/balanco-dre.pdf", "rb")
}
response = requests.post(url, headers=headers, files=files)
print(response.json())
```
